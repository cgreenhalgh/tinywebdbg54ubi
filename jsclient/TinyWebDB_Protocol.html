TinyWebDB Protocol

Store:
  • POST, form '/<NAMESPACE>/storeavalue' , [ 'fmt'='html' - optional] ['jsonp'=<JSONP function> - optional] 'tag' = <tag> (string), 'value' = <value> (string) (x-www-form-urlencoded)
    • GET returns HTML form
    • curl -v -F tag=cmg1 -F value=value1 http://tinywebdbg54ubi.appspot.com/EXAMPLE/storeavalue
  • return ['STORED',<tag>,<value>] (escaped if HTML, surrounded with <JSONP function>(...) if JSONP)
Get:
  • POST form '/<NAMESPACE>/getvalue', x-www-form-urlencoded, [ 'fmt'='html' - optional] ['jsonp'=<JSONP function> - optional] 'tag'=<tag> (string)
    • GET returns HTML form
    • curl -v -F tag=cmg1 http://tinywebdbg54ubi.appspot.com/EXAMPLE/getvalue
  • return ['VALUE',<tag>,<value>] (escaped if HTML, surrounded with <JSONP function>(...) if JSONP)
Delete:
  • POST /deleteentry, entry_key_string (tag present but ignored)
