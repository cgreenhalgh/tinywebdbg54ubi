TinyWebDB Protocol

Store:
  • POST, form '/storeavalue' , [ 'fmt'='html' - optional] 'tag' = <tag> (string), 'value' = <value> (string) (x-www-form-urlencoded)
    • GET returns HTML form
    • curl -v -F tag=cmg1 -F value=value1 http://appinvtinywebdb.appspot.com/storeavalue
  • return ['STORED',<tag>,<value>]
Get:
  • POST form '/getvalue', x-www-form-urlencoded, [ 'fmt'='html' - optional] 'tag'=<tag> (string)
    • GET returns HTML form
    • curl -v -F tag=cmg1 http://appinvtinywebdb.appspot.com/getvalue
  • return ['VALUE',<tag>,<value>] (escaped if HTML)
Delete:
  • POST /deleteentry, entry_key_string (tag present but ignored)
